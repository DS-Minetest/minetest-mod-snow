Snow Biomes

A mod for Minetest which adds snow.

Download .zip Download .tar.gz View on GitHub

Install:

Extract the zip into your minetest mods folder and rename "minetest-mod-snow-master" to "snow".

Forum post: http://minetest.net/forum/viewtopic.php?id=2290

This mod adds snow biomes to minetest. There are nine biome types: (click on one to see the image)

  • Normal
  • Icebergs
  • Icesheet
  • Broken Icesheet
  • Icecave
  • Coast
  • Alpine
  • Snowy
  • Plain

Example map:

Features:

  • You can craft snow blocks with four snowballs.
  • You can craft snow bricks with four snow blocks.
  • Snow and ice melts when near warm blocks such as lava, torches etc.
  • Water freezes over when snow blocks are in contact.
  • 3d snow made possible with nodebox technology!
  • Moss spreads on cobble to make mossy cobble.
  • Christmas trees can be found when digging pine needles.

Uninstall:

Simply delete the folder snow from the mods folder.
